# ClientManagementSystem
Managing of teams by call centre agents

App Stack
* Java 17
* Springboot 2.7.3
* Gradle 6.6.1
* RDBMS: H2
* Lombok

Getting started
* Please make sure you have java >= 17 installed.

Running the app
1. ```gradle build```
2. ```gradle bootRun```

App usage
* Runnif the app http://localhost:8081*
